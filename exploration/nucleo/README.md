# Nucleo 

I bought some ST32F401RE a while back and figured it can be fun 
to get them to run something. Probably nothing meaningful,
but to play a bit with building etc.


## What it does

Nothing, currently it's the base project from CLion (jetbrains) using
STM32Cube, just a blink for now.

## Running

- make sure the udev rules are installed 
    https://docs.platformio.org/en/latest/core/installation/udev-rules.html
- plug in the board
- run the project in clions "upload"
- yay blink
